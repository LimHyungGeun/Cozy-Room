# Cozy-Room
숙박 예약 플랫폼 침해사고
프로젝트 개요
   
본 프로젝트는 취약하게 구현된 숙박 예약 플랫폼을 대상으로 웹 애플리케이션에서 발생할 수 있는 침해 시나리오를 분석하고, 각 취약점의 원인과 영향도, 대응 방안을 정리하는 것을 목적으로 한다.
분석 대상 플랫폼은 객실 예약, 회원가입, 로그인, 리뷰 작성, 관리자 리뷰 검토, 관리자 미디어 관리, 내부 API 조회, 사용자 정보 내보내기 기능 등을 포함하고 있다. 일반적인 숙박 예약 서비스와 유사한 기능을 제공하지만, 일부 관리자 페이지와 API에서 권한 검증이 미흡하고, 사용자 입력값과 업로드 파일에 대한 보안 처리가 부족한 문제가 존재한다.
본 보고서에서는 공격자가 일반 사용자 계정만 보유한 상태에서 시작하여 관리자 기능 일부에 접근하고, 저장형 XSS를 통해 관리자 세션을 탈취한 뒤, 내부 API 정보를 확인하고 최종적으로 전체 사용자 개인정보를 유출하는 흐름을 분석한다.
이를 통해 단일 취약점이 아닌 여러 보안 설정 오류가 연쇄적으로 결합될 경우, 일반 사용자 권한에서 시작된 공격이 대규모 개인정보 유출 사고로 이어질 수 있음을 확인하고자 한다.
